# EchoBridge AI

EchoBridge AI is a web platform where users can talk anonymously, with AI chat and face recognition features.

## Features
- User Login & Signup
- Anonymous Chat
- AI Chat
- Dashboard

## Tech Stack
- HTML, CSS, JavaScript
- Python (Flask)
- SQLite

## How to Run
1. Clone repo
2. Install requirements
3. Run app.py
